Условия и циклы

Конструкция if/else

Выражение if/else проверяет истинность некоторого условия и в зависимости от результатов проверки выполняет определенный код.
Конструкция if/else имеет следующий синтаксис:
  • 1| if (<условие>) {<инструкция 1>} //инструкция 1 выполняется при выполнении условия
  • 2| else {<инструкция 2>} // инструкция 2 выполняется при невыполнении условия
В блоке else также можно добавить условие:
  • 1| else if (<условие 2>) {<инструкция>}
Таких блоков else может быть сколько угодно. Кроме того, они могут вообще не использоваться.
Также возможно соединить сразу несколько условий, используя логические операторы &&, ||, == и др.

Конструкция switch/case

Конструкция switch/case аналогична конструкции if/else, так как позволяет обработать сразу несколько условий. Она имеет следующий вид:
  • 1| switch(<переменная>) {
  • 2|  case <значение 1>: {<инструкция 1>}
  • 3|  case <значение 2>:{<инструкция 2>}
  • 4|  …
  • 5|  case <значение n>: {<инструкция n>}
  • 6|  default: {<инструкция в случае невыполнения ни одной из предыдущих>}
  • 7| }
Здесь в блоке case в случае равенства соответствующего значения значению переменной, указанной в блоке switch, выполняется нужная инструкция. В случае несоответствия ни одному из значений выполняется инструкция из блока default, его использование необязательно.
Возможно использование одной инструкции для нескольких значений:
  • 1| …
  • 2| case <значение 1>:
  • 3| case <значение 2>: {инструкция}
  • 4| …

Тернарная операция

Тернарная операция имеет следующий синтаксис: <первый операнд - условие> ? <второй операнд> : <третий операнд>. Таким образом, в этой операции участвуют сразу три операнда. В зависимости от условия тернарная операция возвращает второй или третий операнд: если условие равно true, то возвращается второй операнд; если условие равно false, то третий. Например:
  • 1| int z = x<y? (x+y) : (x-y);
Здесь результатом тернарной операции является переменная z. Сначала проверяется условие x

Циклы

В языке Java есть следующие виды циклов:
  • for;
  • while;
  • do...while.

Цикл for

Цикл for имеет следующее формальное определение:
  • 1| for (<инициализация счетчика>; <условие>; <изменение счетчика>)
  • 2| {
  • 3|  // действия
  • 4| }

Необязательно указывать все условия при объявлении цикла. Например, можно написать так:
  • 1| int i = 1;
  • 2| for (; ;){
  • 3|  System.out.printf("Квадрат числа %d равен %d \n", i, i * i);
  • 4| }

Определение цикла осталось тем же, только теперь блоки в определении пустые: for (; ;). Теперь нет инициализированной переменной-счетчика, нет условия, поэтому цикл будет работать вечно - бесконечный цикл.
Либо можно опустить ряд блоков:
  • 1| int i = 1;
  • 2| for (; i<9;){
  • 3|  System.out.printf("Квадрат числа %d равен %d \n", i, i * i);
  • 4|  i++;
  • 5| }

Этот пример эквивалентен первому примеру: также есть счетчик, только создан он вне цикла, есть условие выполнения цикла, и есть приращение счетчика уже в самом блоке for.
Цикл for может определять сразу несколько переменных и управлять ими:
  • 1| int n = 10;
  • 2| for(int i=0, j = n - 1; i < j; i++, j--){
  • 3|
  • 4|  System.out.println(i * j);
  • 5| }

Цикл do…while

Цикл do сначала выполняет код цикла, а потом проверяет условие в инструкции while. И пока это условие истинно, цикл повторяется. Например:
  • 1| int j = 7;
  • 2| do{
  • 3|  System.out.println(j);
  • 4|  j--;
  • 5| }
  • 6| while (j > 0);

В данном случае код цикла сработает 7 раз, пока j не окажется равным нулю. Важно отметить, что цикл do гарантирует хотя бы однократное выполнение действий, даже если условие в инструкции while не будет истинно.

Цикл while

Цикл while сразу проверяет истинность некоторого условия, и если условие истинно, то код цикла выполняется:
  • 1| int j = 6;
  • 2| while (j > 0){
  • 3|  System.out.println(j);
  • 4|  j--;
  • 5| }

Операторы continue и break

Оператор break позволяет выйти из цикла в любой его момент, даже если цикл не закончил свою работу. Оператор continue, в свою очередь, переходит к следующей итерации цикла, прерывая текущую.
