Функции, рекурсия

Если переменные и константы хранят некоторые значения, то функции содержат в себе набор операторов, которые выполняют определенные действия.
Общее определение функций выглядит следующим образом:
  • 1| <модификаторы> <тип возвращаемого значения> <название функции> (<параметры>){
  • 2|  <тело функции>
  • 3| }
Модификаторы и параметры необязательны.
По умолчанию главный класс любой программы на Java содержит функцию main, которая служит точкой входа в программу:
  • 1| public static void main(String[] args) {
  • 2|  System.out.println("привет мир!");
  • 3| }
Ключевые слова public и static являются модификаторами. Далее идет тип возвращаемого значения. Ключевое слово void указывает на то, что функция ничего не возвращает.
Затем идут название функции - main и в скобках параметры функции - String[] args. И в фигурные скобки заключено тело функции - все действия, которые она выполняет.
Еще несколько функций:
  •  1 | public class Program{
  •  2 |
  •  3 |  public static void main (String args[]){
  •  4 |  }
  •  5 |  void hello(){
  •  6 |
  •  7 |   System.out.println("Hello");
  •  8 |  }
  •  9 |  void welcome(){
  • 10|
  • 11|   System.out.println("Welcome to Java 10");
  • 12|  }
  • 13| }
Здесь определены две дополнительных функции: hello и welcome, каждая из которых выводит некоторую строку на консоль. Функции определяются внутри класса - в данном случае внутри класса Program, в котором определена функция main.

Но если скомпилировать и запустить данную программу, то на консоли ничего не появится. В примере выше были определены две функции, но нигде не вызывались. По умолчанию в программе Java выполняется только функция main и все ее содержимое. Поэтому, если необходимо, чтобы другие функции тоже выполнялись, их надо вызвать в функции main.
Вызов функции осуществляется в форме:
  • 1| <имя функции>(<аргументы>);
После имени функции указываются скобки, в которых перечисляются аргументы - значения для параметров функции.
Чтобы вызвать в функции main другие функции, которые определены в одном классе с функцией main, они должны иметь модификатор static.
С помощью параметров можно передать в функции различные данные, которые будут использоваться для вычислений. Например:
  • 1| static void sum(int x, int y){
  • 2|  int z = x + y;
  • 3|  System.out.println(z);
  • 4| }
Данная функция принимает два параметра - два числа, складывает их и выводит их сумму на консоль, а при вызове этой функции в программе необходимо передать на место параметров значения, которые соответствуют типу параметра.

Параметры переменной длины

Функция может принимать параметры переменной длины одного типа. Например, необходимо передать в функцию набор чисел и вычислить их сумму, но точно неизвестно, сколько именно чисел будет передано. Параметры переменной длины позволяют решить эту задачу:
  •  1 | public class Program{
  •  2 |
  •  3 |  public static void main (String args[]){
  •  4 |   sum(1, 2, 3); // 6
  •  5 |   sum(1, 2, 3, 4, 5); // 15
  •  6 |   sum(); // 0
  •  7 |  }
  •  8 |  static void sum(int ...nums){
  •  9 |
  • 10|   int result =0;
  • 11|   for(int n: nums)
  • 12|    result += n;
  • 13|   System.out.println(result);
  • 14|  }
  • 15| }
Многоточие перед названием параметра int ...nums указывает на то, что он будет необязательным и будет представлять массив. Можно передать в функцию sum одно число, несколько чисел, а можно вообще не передавать никаких параметров. Причем, если необходимо передать несколько параметров, то необязательный параметр должен указываться в конце:
  •  1 | public static void main(String[] args) {
  •  2 |
  •  3 |  sum("Welcome!", 20,10);
  •  4 |  sum("Hello World!");
  •  5 | }
  •  6 | static void sum(String message, int ...nums){
  •  7 |
  •  8 |  System.out.println(message);
  •  9 |  int result =0;
  • 10|  for(int x:nums)
  • 11|   result += x;
  • 12|  System.out.println(result);
  • 13| }

Рекурсия

Главное отличие рекурсивных функций от обычных состоит в том, что рекурсивная функция может вызывать саму себя.
Например, функция, которая вычисляет факториал числа:
  • 1| static int factorial(int x){
  • 2|
  • 3|  if (x == 1){
  • 4|
  • 5|   return 1;
  • 6|  }
  • 7|  return x * factorial(x - 1);
  • 8| }
Сначала проверяется условие: если вводимое число не равно 1, то оно умножается на результат этой же функции, в которую в качестве параметра передается число x-1. То есть происходит рекурсивный спуск. И так дальше, пока не будет достигнут момент, когда значение параметра не будет равно единице.
Рекурсивная функция обязательно должна иметь некоторый базовый вариант, который использует оператор return и который помещается в начале функции. В случае с факториалом это if (x == 1) return 1;.

Все рекурсивные вызовы должны обращаться к подфункциям, которые в конечном счете сходятся к базовому варианту. Так, при передаче в функцию положительного числа при дальнейших рекурсивных вызовах подфункций в них будет передаваться каждый раз число, меньшее на единицу. В конце концов будет достигнута ситуация, когда число будет равно 1, и будет использован базовый вариант.
